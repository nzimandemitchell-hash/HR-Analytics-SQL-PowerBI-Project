# HR-Analytics-SQL-PowerBI-Project
End-to-end HR Analytics project using SQL Server and Power BI.


## Project Overview

This project demonstrates an end-to-end data analytics workflow using SQL Server and Power BI to transform raw HR employee data into actionable business insights.

The project follows a complete analytics process, starting with data cleaning and preparation, moving into exploratory data analysis (EDA), and ending with the development of an interactive Power BI dashboard for business decision-making.

---

 Project Objectives

* Clean and prepare raw HR data for analysis.
* Explore workforce trends and employee demographics.
* Identify patterns in salaries, departments, locations, and employment status.
* Build an interactive dashboard that allows users to explore HR metrics dynamically.

---

## Tools and Technologies Used

* SQL Server
* T-SQL
* Power BI
* DAX



## Project Workflow

 1. Data Cleaning in SQL Server

The raw dataset contained inconsistencies and required preprocessing before analysis.

Data cleaning tasks included:

* Removing duplicate records
* Handling missing values
* Correcting invalid age values
* Standardizing text formatting
* Converting data types
* Improving overall data quality and consistency



 2. Exploratory Data Analysis (EDA)

After cleaning the dataset, SQL queries were used to explore employee trends and generate business insights.

Analysis included:

* Employee distribution by department
* Average salary by department
* Workforce distribution by city
* Employment status analysis
* Job title distribution
* Salary trends across age groups
* Highest-paying departments

Advanced SQL concepts used:

* Common Table Expressions (CTEs)
* Window Functions
* Ranking Functions
* Aggregate Functions
* CASE Statements

---

 3. Dashboard Development in Power BI

An interactive HR Analytics dashboard was developed to visualize key workforce metrics.

-- KPI Cards

* Total Employees
* Total Payroll Cost
* Active Employees
* Resigned Employees

-- Visualizations

* Employees by Department
* Payroll by Department
* Employee Distribution by City
* Employment Status Breakdown
* Salary Trends by Age
* Employee Distribution by Job Title

-- Interactive Features

* Employment Status slicer
* Department slicer
* City slicer




## Key Business Insights

* Operations and Marketing employ the largest proportion of employees.
* Payroll costs vary significantly between departments.
* Workforce distribution differs across major South African cities.
* Employment status metrics provide insight into employee retention and turnover.
* Certain job titles make up a significant proportion of the workforce.
